# 🏋️‍♂️ MRCoach Fitness: WhatsApp Automation & CRM

**MRCoach Fitness** is a professional-grade, automated lead management and CRM system designed to streamline fitness coaching operations via WhatsApp. It eliminates manual data entry by using an intelligent bot to capture, qualify, and organize leads directly into a high-end SaaS dashboard.

### 🚀 Key Features

*   **⚡ Automated Client Onboarding**: A smart, multi-step conversation flow that captures client goals, location, and requirements instantly.
*   **🧘 Intelligent Coach Recruitment**: A robust parsing system for fitness coaches to register their specialization, experience, and city—with automatic flyer delivery upon completion.
*   **📊 Pro SaaS Admin Dashboard**: A sleek, real-time control center featuring:
    *   **KPI Performance Section**: Track Total, Pending, Converted, and Today's Leads at a glance.
    *   **Advanced Grid Filtering**: Search and filter by Name, Phone, City, or Status in a high-speed grid layout.
    *   **Dynamic Analytics**: Location-based dropdowns show exactly how many leads are coming from each city.
*   **💳 Integrated Payments & Tracking**: Built-in support for **Razorpay** payment links and automated order tracking for fitness plans.
*   **🛡️ Group-Safe Technology**: Smart filtering ensures the bot only responds to private inquiries, keeping your personal WhatsApp groups clean.
*   **📥 Professional Data Export**: Export lead data to **Excel** or **PDF** with a single click for offline reporting.

### 🛠️ Tech Stack
*   **Backend**: Node.js / Express.js
*   **Automation**: `whatsapp-web.js` (Private Chat exclusive)
*   **Database**: SQLite3 (Clean, portable, and fast)
*   **UI/Frontend**: Vanilla CSS / JavaScript (SaaS Material Design)

### 🚀 Getting Started

1.  **Clone the project**:
    ```bash
    git clone https://github.com/2kamalesh0809-ship-it/whatsapp-Bot.git
    cd whatsapp-Bot
    ```
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Setup Environment Variables**:
    Create a `.env` file with `BRAND_NAME`, `RAZORPAY_KEY_ID`, and `RAZORPAY_KEY_SECRET`.
4.  **Run the bot**:
    ```bash
    npm start
    ```
